# Personal Assistant CLI
Консольний застосунок "Персональний помічник" для збереження контактів.
Проєкт виконаний в рамках домашніх завдань курсу Python Core.

## Завдання 1 — Віртуальне середовище
Для створення та управління віртуальним середовищем використано Poetry.

### Версія Python
Проєкт використовує:
Python 3.12

Версія Python зафіксована у файлі:
pyproject.toml

### Структура проєкту
goit-ds-hw-01/
│
├── pyproject.toml
├── poetry.lock
├── README.md
│
└── src/
    └── goit-ds-hw-01/
        └── contact_bot.py

Структура src використовується відповідно до рекомендацій Python packaging.

## Встановлення залежностей

### Встановлення середовища:
```
poetry install
```
### Активація середовища:
```
poetry shell
```
або запуск без активації:
```
poetry run contact-bot
```
